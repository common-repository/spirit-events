Simple event calendar. 

== Description ==
Spirit events offers a simple way how to handle events in WordPress.

Key functionalities are:

- widget showing 5 upcoming events

- calendar view

- live stream support 

- UI can be customized through the plugin settings

- Google Maps support - API key needed

== Installation ==

1. Upload the full spirit-events directory into your wp-content/plugins directory.
2. In WordPress select Plugins from your sidebar menu and activate the Spirit events plugin.
